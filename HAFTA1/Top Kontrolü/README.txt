# Proje Adı: Unity Küp Hareket Ettirme 

## İçerik
Bu proje, Unity kullanarak geliştirilen temel bir oyun projesidir. Oyuncu, klavye tuşları ile bir objeyi yönlendirebilir ve fizik motoru kullanarak hareket ettirebilir.

## Özellikler
- **Klavye Kontrolleri:** Objeyi Yön Tuşları veya WASD ile hareket ettirme
- **Fizik Motoru:** Yerçekimi ve hareket fiziklerini destekler

## Kurulum
1. **Unity'yi Yükleyin** (Unity Hub kullanarak en güncel sürüm)
2. **Projeyi Klonlayın**
   ```bash
   git clone https://github.com/emineecekiyak/UNITY-HAFTALIK-ODEVLER.git
   ```
3. **Unity ile Projeyi Açın**
4. **Play Tuşuna Basarak Test Edin**

## Kullanım
- Oyunu başlatmak için **Play** butonuna basın.
- Yön tuşları veya **WASD** kullanarak objeyi hareket ettirin.
- Kameranın objeye odaklandığını kontrol edin.

## Katkıda Bulunma
1. Forklayın
2. Yeni bir branch oluşturun: `git checkout -b yeni-ozellik`
